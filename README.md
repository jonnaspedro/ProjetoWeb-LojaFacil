# ProjetoWeb-LojaFacil
Sobre LojaFácil é um aplicativo web para pequenas e médias lojas de roupas, permitindo cadastrar produtos, controlar estoque, registrar vendas e gerar relatórios simples. O sistema também inclui CRUDs de fornecedores, funcionários, categorias de produtos e marcas, ampliando o gerenciamento e facilitando a organização da loja.
